# linebot_manageUrl
一個管理大量連結網站的LINEBOT APP，運用LINE提供的官方帳號webhook開發者功能，製作出適合自己管理網站連結的app

## Main Feature
1. 在特定群組傳訊，如「#docekr https://docker.com」("{tag} {url}")，資訊會儲存在雲端Google Sheet
2. 在官方帳號中可以依據tag尋找，需輸入「凱莉找..」，找的後方直接填寫資訊，不需要有空白格。
